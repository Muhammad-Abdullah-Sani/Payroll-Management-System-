# Payroll Management System in C++

A C++ console-based Payroll Management System that allows admins and employees to manage payroll operations efficiently. The system tracks employee data, attendance, and calculates payroll, with data stored in text files for persistence.

---

## Features

### Admin and User Roles
- **Admin:** Add, edit, delete employee records, manage payroll, and view reports.  
- **User:** View personal details, attendance, and salary information.

### Employee Data Management
- Store employee information including name, ID, position, and salary.  
- Edit or update employee records easily.

### Attendance Tracking
- Mark attendance for employees.  
- Automatically integrate attendance into payroll calculation.

### Payroll Calculation
- Computes salary based on base pay, attendance, and any deductions or bonuses.  
- Generates simple salary reports for employees.

### File-based Persistence
- Employee and payroll data are stored in text files (`.txt`) to maintain data between sessions.

### Interactive Menu System
- Separate menus for admins and employees for easier navigation.  
- Console-based interface with clear prompts.
